# Croak Webpage

Landing page for Croak, a smart water quality monitoring product.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Requirements

- Node.js
- npm

Check versions:

`node -v`
`npm -v`

## Run Locally

Install dependencies:

`npm install`

Start the dev server:

`npm run dev`

Open the local URL shown in the terminal, usually `http://localhost:5173`.

## Build

Create a production build:

`npm run build`

## Team Workflow

1. Pull latest changes
2. Create a branch
3. Make changes
4. Test locally
5. Commit clearly
6. Push branch
7. Open a PR or share for review

Suggested branch names:

`feature/hero-animation`
`feature/navbar-cleanup`
`fix/footer-links`
`refactor/faq-structure`

Suggested commit messages:

`feat: update hero section layout`
`fix: correct footer links`
`refactor: simplify FAQ component`
`style: adjust spacing and typography`
`chore: clean project setup`

## Rules

- Do not commit `node_modules`
- Do not commit `dist`
- Test locally before pushing
- Keep commits focused
- Avoid adding dependencies unless necessary
- Prefer improving existing components instead of duplicating code

## Notes

- `dist/` is generated by `npm run build`
- `node_modules/` is local only
- some Tailwind v4 directives may show editor warnings in VS Code even if the project builds correctly
